Principais métodos de Inteligência Artificial

Existem muitos métodos utilizados na programação de NPC’s, porém os principais são: Deep Learning e Algoritmos Genéticos. Cada técnica é programada para realizar ações durante o jogo. Para isso, há algoritmos responsáveis por monitorar o cenário, guardando informações importantes para o bom funcionamento das dinâmicas dentro do jogo.

Deep Learning

Deep learning, uma subcategoria muito utilizada no campo da IA, aplica redes neurais em camadas de profundidade, dando importância a comunicação entre neurônios, que possuem a habilidade de se modificarem quando necessário. De acordo com Data Science Academy (2019, n.p), “O aprendizado profundo é baseado no conceito de redes neurais artificiais, ou sistemas computacionais que imitam a maneira como o cérebro humano funciona.”

As RNA’s (redes neurais artificiais), conforme Araújo (2017), são técnicas que pertencem a um agrupamento de estudos denominado Inteligência Computacional. Elas são desenvolvidas através de modelos matemáticos que simulam as redes neurais de seres inteligentes, conseguindo assim a habilidade de aprendizagem através de experiências passadas. Na Figura a seguir pode-se observar a representação de um neurônio artificial, criado a partir de cálculos matemáticos.



De acordo com Facure (2017), o neurônio artificial é uma soma ponderada de n inputs que, após passar por determinada função, gera um resultado e passa ao próximo neurônio conectado a ele, como mostra a Figura anterior. Unindo vários neurônios artificias, é possível forma uma rede neural, capaz de aprender com os vários resultados que ela obtiver de cada neurônio. Na Figura abaixo é possível observar um exemplo de rede neural simples e várias redes neurais unidas, que formam uma Deep Learning Neural Network.



Algoritmos deep learning, segundo Sejnowski (2019), dependem da forma em que os neurônios estão se comunicando entre eles, para se modificarem a partir das experiências. Essa complexidade é transformada em padrões de atividade, denominados “ingredientes da inteligência”.

Outro aspecto que compõe a deep learning é a extração de recursos. A extração ocorre por meio do cientista de dados ou engenheiro de IA, que ficam responsáveis pelo algoritmo que constrói automaticamente esses recursos, a fim de utilizá-los para treinamento para futuras máquinas (DATA SCIENCE ACADEMY, 2019). Abaixo estão listadas algumas arquiteturas de redes neurais:

  • Redes Multilayer Perceptrons: classificação binária;
  • Redes Neurais Convolucionais: classificação de imagens;
  • Redes Neurais Recorrentes: processamento de dados sequenciais;
  • Long Short-Term Memory (LSTM): Preservar o erro, podendo ser copiado por tempo e camadas. Solução do problema vanishing gradiente;
  • Redes de Hopfield: Armazenamento de memória e construção de interpretações de entrada sensorial;
  • Máquinas de Boltzmann: Representar e resolver problemas combinatórios difíceis;
  • Deep Belief Network: Reconhecer, agrupar e gerar imagens, sequências de vídeos e dados de captura de movimento;
  • Deep Auto-Encoders: Modelagem de tópicos ou modelagem estatística de tópicos abstratos em coleção de documentos;
  • Generative Adversarial Network: Reprodução de qualquer distribuição de dados;
  • Deep Neural Network Capsules: Modelar relações hierárquicas dentro da representação do conhecimento interno de uma rede neural.

As arquiteturas citadas acima são usadas para reconhecimento de padrões, detecção de anomalias e previsões futuras. São muito aplicadas na medicina, no trânsito, na segurança, na robótica e até em tratamentos para pessoas com deficiências visuais (DATA SCIENCE ACADEMY, 2019).

Algoritmos Genéticos

Os algoritmos genéticos, conforme Karlsson (2005, p. 33), “[...] usam uma abordagem inspirada no processo de seleção natural e tentam imitar esse processo para evoluir até encontrar uma solução próxima à ótima para o problema”. Esses algoritmos tendem a responder de diversas formas, de acordo com as ações realizadas durante um jogo que o possui. Na Figura a seguir é representada a estrutura básica de um algoritmo genético.



Esses algoritmos funcionam como na Biologia: para evoluir um personagem, algumas propriedades do agente servirão como os genes. Cada gene possui um valor, denominado alelo. Os alelos são responsáveis por definir as características que o personagem terá ao evoluir como, por exemplo, a cor dos olhos, da pele e do cabelo. A junção de vários genes forma um cromossomo, que representará um indivíduo (KARLSSON, 2005).

Esse tipo de algoritmo necessita de espécies iniciais para criar outras espécies a partir destas. De acordo com Oliveira (2018), pode-se ter n recombinações entre os pontos, com vários cortes aleatórios que definem seções, que são compartilhadas entre descendentes dentro de um mesmo corte. Karlsson (2005) comenta que o algoritmo funciona como na realidade, onde são necessários dois indivíduos para reproduzir um novo. Com base nisso, o algoritmo genético irá permitir que as espécies se reproduzam, até encontrar algum gatilho que o faça parar a reprodução.

Segundo Miranda (2011), os Algoritmos Genéticos possuem quatro operações simples para seu funcionamento. São elas: cálculo da aptidão; seleção; cruzamento e mutação. Abaixo há uma lista contendo as operações básicas de AG, juntamente com suas respectivas especificações:

  • Cálculo de aptidão: Um indivíduo passa por uma ferramenta de análise de desempenho e, após encontrar as medidas que são obtidas como resultado, são jogadas em uma função para obtenção das aptidões do indivíduo, sendo possível classificá-los conforme os resultados;
  • Seleção: Nessa fase, os indivíduos são selecionados para gerar uma nova geração. A probabilidade de um indivíduo x ser escolhido é proporcional à sua aptidão. Para fazer esta seleção, o método aplicado é a “Amostragem Estocástica Universal”, que possibilita pegar os indivíduos mais aptos distribuídos dentro de um círculo;
  • Cruzamento: O cruzamento acontece da seguinte forma: é gerada uma segunda lista, denominada lista de parceiros, a partir da primeira lista de indivíduos selecionados. Após isso, cada indivíduo da primeira lista é cruzado com um indivíduo que ocupa o mesmo espaço na segunda lista. Os cromossomos de ambos os indivíduos são separados a partir de um ponto de corte e, para gerar um novo indivíduo, pega-se a metade inicial do primeiro indivíduo e a metade final do segundo indivíduo e as junta, formando assim a próxima geração;
  • Mutação: Esta função é utilizada para diminuir as chances de convergência na criação da nova geração. Para não ter erro, ela pega um indivíduo qualquer e altera um dos genes dele, que é escolhido aleatoriamente.

De acordo com Karlsson (2005, p. 35), “Algoritmos genéticos são úteis quando é difícil prever as interações entre o conjunto de parâmetros que regula o comportamento dos agentes e o mundo do jogo.”.
